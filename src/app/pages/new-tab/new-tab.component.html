A Chrome extension I made for personal use to replace the boring default new tab homepage. In the spirit of iGoogle, it displays the browser's location, time, weather, and any severe weather alerts and automatically responds to changes in location and timezone. The background color is the current time as a hex code.

The webapp itself is built in AngularJS 1.6, and service calls are routed through Python-based AWS Lambda functions (for throttling and to prevent leaking API keys).

The original idea was inspired by What colour is it?. Geolocation information is from the Google Maps API. Weather comes from the Dark Sky Forecast API. Weather icons are erikflowers/weather-icons. Weather alerts are from the National Weather Service API.
